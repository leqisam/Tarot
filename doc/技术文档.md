# 星语塔罗 - 技术文档

## 项目概述

星语塔罗是一个基于 Web 的塔罗牌占卜应用，采用纯前端技术栈实现，接入 AI 大模型提供智能解读服务。

## 技术栈

### 核心技术

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript (ES6+)** - 交互逻辑
- **TailwindCSS** - 原子化 CSS 框架（通过 CDN 引入）

### 第三方库与服务

- **TailwindCSS CDN** - 快速样式开发
- **Google Fonts** - Noto Serif SC 和 Cinzel 字体
- **火山引擎 DeepSeek-V3** - AI 智能解读服务

### API 调用示例

```javascript
// 获取 AI 解读
async function getAIInterpretation(question, cards, spreadType) {
    const response = await fetch(AI_CONFIG.apiUrl, {
        method: 'POST',
        headers: {
            'Authorization': `Bearer ${AI_CONFIG.apiKey}`,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({
            model: AI_CONFIG.model,
            stream: false,
            input: [{
                role: 'user',
                content: [{
                    type: 'input_text',
                    text: buildPrompt(question, cards, spreadType)
                }]
            }]
        })
    });
    
    const data = await response.json();
    return data.output.text;
}
```

### 提示词构建

AI 解读的提示词包含以下信息：
- 用户问题
- 牌阵类型和描述
- 每张牌的详细信息（名称、正逆位、关键词、含义、建议）
- 解读要求（整体氛围、逐张解读、牌间关联、行动建议）

### 备用方案

当 API 不可用时，自动切换至本地解读生成：
- 分析牌面元素分布
- 统计正逆位比例
- 生成整体主题描述
- 组合单张牌解读

## 文件说明

### HTML 文件

#### index.html - 首页

- **功能**：应用入口页面，展示 Hero 区域、快速入口、最近占卜记录
- **主要特性**：
  - 星空背景动画
  - 响应式导航栏
  - 每日一牌快速入口
  - 最近占卜记录展示

#### divination.html - 占卜流程页面

- **功能**：完整的多牌阵占卜流程，支持AI智能解读
- **流程步骤**：
  1. 问题输入（5-200字限制）
  2. 牌阵选择（单张牌/三张牌/凯尔特十字/关系牌阵）
  3. 准备阶段（深呼吸提示）
  4. 洗牌阶段（动画效果）
  5. 切牌阶段（用户交互）
  6. 抽卡阶段（根据牌阵抽取对应数量卡牌）
  7. 翻牌阶段（3D翻转动画，按牌阵位置排列）
  8. 结果展示（AI智能解读 + 单张牌详情）

- **支持的牌阵**：
  | 牌阵 | 卡牌数 | 适用场景 |
  |-----|-------|---------|
  | 单张牌 | 1 | 每日运势、简单问题 |
  | 三张牌-时间之流 | 3 | 过去-现在-未来 |
  | 三张牌-问题分析 | 3 | 问题-障碍-建议 |
  | 凯尔特十字 | 10 | 深度全面分析 |
  | 关系牌阵 | 5 | 感情关系分析 |

#### profile.html - 个人中心页面

- **功能**：用户数据管理和设置
- **主要特性**：
  - 占卜历史记录列表
  - 搜索和筛选功能
  - 数据统计（总次数、连续天数、最常出现的牌）
  - 设置页面（动画开关、背景开关）
  - 数据导出和清除

## 响应式设计

### 断点设置

- **移动端**：< 640px
- **平板端**：640px - 1024px
- **桌面端**：> 1024px

### 适配策略

- 使用 TailwindCSS 的响应式前缀（sm:, md:, lg:）
- 卡牌尺寸在移动端缩小（120px → 100px）
- 导航栏在移动端变为汉堡菜单
- 布局从多列变为单列

## 动画效果

### 星空背景

- 随机生成 100 颗星星
- 每颗星星有随机的位置、大小和闪烁延迟
- 使用 CSS 动画实现闪烁效果

### 卡牌动画

- **洗牌**：左右摇摆动画（@keyframes shuffleCard）
- **翻牌**：3D Y轴旋转 180 度
- **逆位**：额外旋转 180 度
- **悬停**：放大和阴影效果

### 页面过渡

- 步骤切换使用 hidden/display 控制
- 元素进入使用 fadeIn 和 slideUp 动画

## 开发规范

### 命名规范

- **CSS 类名**：使用 kebab-case（如 `.tarot-card`）
- **JavaScript 变量**：使用 camelCase（如 `drawnCard`）
- **文件名**：使用小写和连字符（如 `common.css`）

### 代码组织

- 公共样式和脚本放在 `assets` 目录
- 页面特定代码放在各自 HTML 文件的 `<script>` 标签中
- 使用语义化 HTML 标签
