# 星语塔罗 - 技术文档

## 项目概述

星语塔罗是一个基于 Web 的塔罗牌占卜应用，采用纯前端技术栈实现，无需后端服务器即可运行。

## 技术栈

### 核心技术

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript (ES6+)** - 交互逻辑
- **TailwindCSS** - 原子化 CSS 框架（通过 CDN 引入）

### 第三方库

- **TailwindCSS CDN** - 快速样式开发
- **Google Fonts** - Noto Serif SC 和 Cinzel 字体

## 文件说明

### HTML 文件

#### index.html - 首页

- **功能**：应用入口页面，展示 Hero 区域、快速入口、最近占卜记录
- **主要特性**：
  - 星空背景动画
  - 响应式导航栏
  - 每日一牌快速入口
  - 最近占卜记录展示

#### divination.html - 占卜流程页面

- **功能**：完整的单张牌占卜流程
- **流程步骤**：
  1. 问题输入（5-200字限制）
  2. 准备阶段（深呼吸提示）
  3. 洗牌阶段（动画效果）
  4. 切牌阶段（用户交互）
  5. 抽卡阶段（随机抽取）
  6. 翻牌阶段（3D翻转动画）
  7. 结果展示（牌意解读）

#### profile.html - 个人中心页面

- **功能**：用户数据管理和设置
- **主要特性**：
  - 占卜历史记录列表
  - 搜索和筛选功能
  - 数据统计（总次数、连续天数、最常出现的牌）
  - 设置页面（动画开关、背景开关）
  - 数据导出和清除

### CSS 文件

#### common.css - 公共样式

定义了以下核心样式：

**CSS 变量**

```css
:root {
    --tarot-dark: #0a0a1a;        /* 深色背景 */
    --tarot-purple: #1a1a3e;      /* 主题紫色 */
    --tarot-gold: #d4af37;        /* 金色强调 */
    --tarot-mystic: #6b4c9a;      /* 神秘紫 */
}
```

**核心组件样式**

- `.stars` - 星空背景容器
- `.star` - 星星元素（带动画）
- `.glass` - 玻璃态效果（毛玻璃 + 边框）
- `.tarot-card` - 塔罗牌基础样式（3D翻转）
- `.card-face` - 卡牌正反面
- `.card-back` - 卡牌背面样式
- `.card-front` - 卡牌正面样式

**动画效果**

- `@keyframes twinkle` - 星星闪烁
- `@keyframes float` - 悬浮动画
- `@keyframes glow` - 发光效果
- `@keyframes fadeIn` - 淡入效果
- `@keyframes slideUp` - 上滑进入

## 响应式设计

### 断点设置

- **移动端**：< 640px
- **平板端**：640px - 1024px
- **桌面端**：> 1024px

### 适配策略

- 使用 TailwindCSS 的响应式前缀（sm:, md:, lg:）
- 卡牌尺寸在移动端缩小（120px → 100px）
- 导航栏在移动端变为汉堡菜单
- 布局从多列变为单列

## 动画效果

### 星空背景

- 随机生成 100 颗星星
- 每颗星星有随机的位置、大小和闪烁延迟
- 使用 CSS 动画实现闪烁效果

### 卡牌动画

- **洗牌**：左右摇摆动画（@keyframes shuffleCard）
- **翻牌**：3D Y轴旋转 180 度
- **逆位**：额外旋转 180 度
- **悬停**：放大和阴影效果

### 页面过渡

- 步骤切换使用 hidden/display 控制
- 元素进入使用 fadeIn 和 slideUp 动画

## 开发规范

### 命名规范

- **CSS 类名**：使用 kebab-case（如 `.tarot-card`）
- **JavaScript 变量**：使用 camelCase（如 `drawnCard`）
- **文件名**：使用小写和连字符（如 `common.css`）

### 代码组织

- 公共样式和脚本放在 `assets` 目录
- 页面特定代码放在各自 HTML 文件的 `<script>` 标签中
- 使用语义化 HTML 标签

## 部署说明

### 本地开发

```bash
# 进入项目根目录
cd d:\ai学习\study\Tarot

# 启动本地服务器
python -m http.server 8082

# 访问地址
http://localhost:8082/src/index.html
```

### 生产部署

1. 将 `src` 目录下的所有文件上传到 Web 服务器
2. 确保服务器支持静态文件服务
3. 可选：将 TailwindCSS CDN 替换为本地构建版本
